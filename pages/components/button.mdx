# 按钮 Button

这是什么？点一下

### 大小

使用 `size` 调整大小

<br/>

<Button size="ty">小小</Button>
<Button size="sm">小</Button>
<Button>正常</Button>
<Button size="lg">大</Button>
<Button size="xl">大大</Button>

```tsx copy
import { Button } from "mmixel-ui"

export default function App() {
  return (
    <Button size="ty">小小</Button>
    <Button size="sm">小</Button>
    <Button>正常</Button>
    <Button size="lg">大</Button>
    <Button size="xl">大大</Button>
  )
}
```

### 样式

使用 `type` 调整样式

<br/>

<Button type="primary">Primary</Button>
<Button type="secondary">Secondary</Button>
<Button type="ghost">Ghost</Button>

```tsx copy
import { Button } from "mmixel-ui"

export default function App() {
  return (
    <Button type="primary">Primary</Button>
    <Button type="secondary">Secondary</Button>
  )
}
```

### 形状

使用 `rounded` 或 `square` 调整形状

<br/>

<Button>不圆</Button>
<Button rounded>圆润</Button>
<Button square>方</Button>
<Button square rounded>原</Button>

```tsx copy
import { Button } from "mmixel-ui"

export default function App() {
  return (
    <Button>不圆</Button>
    <Button rounded>圆润</Button>
    <Button square>方</Button>
    <Button square rounded>原</Button>
  )
}
```

### 按钮组

<br/>

<ButtonGroup>
  <Button>富强</Button>
  <Button type="primary">民主</Button>
  <Button type="secondary">文明</Button>
  <Button>和谐</Button>
</ButtonGroup>

<ButtonGroup direction="ver" rounded>
  <Button>富强</Button>
  <Button type="primary">民主</Button>
  <Button type="secondary">文明</Button>
  <Button>和谐</Button>
</ButtonGroup>

### Button Props

| Prop     | Type                                               | Default     | Desc           |
| :------- | :------------------------------------------------- | :---------- | :------------- |
| type     | `"default" \| "primary" \| "secondary" \| "ghost"` | `"default"` | 按钮的样式     |
| size     | `"ty"\| "sm"   \| "md"\| "lg"\| "xl"`              | `"md"`      | 按钮的大小     |
| rounded  | `boolean`                                          | `false`     | 按钮是否圆角   |
| square   | `boolean`                                          | `false`     | 按钮是否正方形 |
| disabled | `boolean`                                          | `false`     | 按钮禁用状态   |
| loading  | `boolean`                                          | `false`     | 按钮加载状态   |
| icon     | `ReactNode`                                        | `undefined` | 按钮图标       |

### ButtonGroup Props

| Prop      | Type             | Default | Desc           |
| :-------- | :--------------- | :------ | :------------- |
| direction | `"hor" \| "ver"` | `"hor"` | 按钮组的方向   |
| rounded   | `boolean`        | `false` | 按钮组是否圆角 |
