# 下拉菜单 Dropdown

将一些操作归类到一起

### 默认

最原始的菜单

<br/>

<Dropdown trigger={<Button>操作</Button>}>
  <div className="p-2 grid bg-white">
    <Link href="" className="px-6 py-2 hover:bg-gray-100">点赞</Link>
    <Link href="" className="px-6 py-2 hover:bg-gray-100">投币</Link>
    <Link href="" className="px-6 py-2 hover:bg-gray-100">收藏</Link>
  </div>
</Dropdown>

```tsx copy
import { Dropdown } from "mmixel-ui"

export default function App() {
  return (
    <Dropdown trigger={<Button>操作</Button>}>
      <div className="p-2 grid bg-white">
        <Link href="" className="px-6 py-2 hover:bg-gray-100">点赞</Link>
        <Link href="" className="px-6 py-2 hover:bg-gray-100">投币</Link>
        <Link href="" className="px-6 py-2 hover:bg-gray-100">收藏</Link>
      </div>
    </Dropdown>
  )
}
```

### 触发方式

使用 `listen` 改变触发方式

<br/>

<Dropdown trigger={<Button>点我</Button>} listen="click">
  <div className="p-2 grid bg-white">
    <Link href="" className="px-6 py-2 hover:bg-gray-100">点赞</Link>
    <Link href="" className="px-6 py-2 hover:bg-gray-100">投币</Link>
    <Link href="" className="px-6 py-2 hover:bg-gray-100">收藏</Link>
  </div>
</Dropdown>

```tsx copy
import { Dropdown } from "mmixel-ui"

export default function App() {
  return (
    <Dropdown trigger={<Button>点我</Button>} listen="click">
      <div className="p-2 grid bg-white">

        <Link href="" className="px-6 py-2 hover:bg-gray-100">收藏</Link>
      </div>
    </Dropdown>
  )
}
```

### 方向

使用 `direction` 改变菜单方向

<br/>

<Dropdown trigger={<Button>上面！</Button>} direction="top">
  <div className="p-2 grid bg-white">
    <Link href="" className="px-6 py-2 hover:bg-gray-100">点赞</Link>
    <Link href="" className="px-6 py-2 hover:bg-gray-100">投币</Link>
    <Link href="" className="px-6 py-2 hover:bg-gray-100">收藏</Link>
  </div>
</Dropdown>

```tsx copy
import { Dropdown } from "mmixel-ui"

export default function App() {
  return (
    <Dropdown trigger={<Button>上面！</Button>} direction="top">
      <div className="p-2 grid bg-white">
        <Link href="" className="px-6 py-2 hover:bg-gray-100">点赞</Link>
        <Link href="" className="px-6 py-2 hover:bg-gray-100">投币</Link>
        <Link href="" className="px-6 py-2 hover:bg-gray-100">收藏</Link>
      </div>
    </Dropdown>
  )
}
```

### Dropdown Props

| Prop      | Type                 | Default     | Desc         |
| :-------- | :------------------- | :---------- | :----------- |
| trigger   | `ReactElement`       | `undefined` | 菜单的触发器 |
| listen    | `"hover" \| "click"` | `"hover"`   | 菜单触发方式 |
| direction | `"top" \| "bottom"`  | `"bottom"`  | 菜单出现朝向 |
