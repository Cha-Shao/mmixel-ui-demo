# 标题 Title

做这个只是因为我不知道什么大小应该用h几

### 大小

使用 `size` 调整大小

<br/>

<Title size="ty">小小</Title>
<Title size="sm">小</Title>
<Title>不小</Title>
<Title size="lg">大</Title>
<Title size="xl">大大</Title>

```tsx copy
import { Title } from "mmixel-ui"

export default function App() {
  return (<>
    <Title size="ty">小小</Title>
    <Title size="sm">小</Title>
    <Title>不小</Title>
    <Title size="lg">大</Title>
    <Title size="xl">大大</Title>
  </>)
}
```

### 超级标题

为什么是这样的名字

<BigTitle title={["头号", "玩家"]} />

<div className="my-24" />

<BigTitle
  title={["假面", "骑士"]}

```tsx copy
import { Title } from "mmixel-ui"

export default function App() {
  return (<>
    <BigTitle title={["头号", "玩家"]} />
    <div className="my-24" />
    <BigTitle
      title={["假面", "骑士"]}
      titleColor={["#ff479f","#92f60f"]}
      blurColor={["#83e701", "#ff479f"]}/>
  </>)
}
```  

### Title Props

| Prop | Type                                   | Default | Desc       |
| :--- | :------------------------------------- | :------ | :--------- |
| size | `"ty" \| "sm" \| "md" \| "lg" \| "xl"` | `"md"`  | 标题的大小 |

### BigTitle Props

| Prop       | Type               | Default                  | Desc         |
| :--------- | :----------------- | :----------------------- | :----------- |
| title      | `[string, string]` | `[undefined, undefined]` | 标题内容     |
| titleColor | `[string, string]` | `[undefined, undefined]` | 标题颜色     |
| blurColor  | `[string, string]` | `[undefined, undefined]` | 背景模糊颜色 |
