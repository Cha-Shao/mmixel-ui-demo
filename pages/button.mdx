# 按钮 Button

这是什么？点一下

import Button from '../components/Button'

### 常规

使用 `size` 调整按钮大小

<Button size="ty">小小</Button>
<Button size="sm">小</Button>
<Button>正常</Button>
<Button size="lg">大</Button>
<Button size="xl">大大</Button>

```js copy
import Button from "mmixel-ui"

export default function App() {
  return (
    <Button size="ty">小小</Button>
    <Button size="sm">小</Button>
    <Button>正常</Button>
    <Button size="lg">大</Button>
    <Button size="xl">大大</Button>
  )
}
```

### 样式

使用 `type` 调整按钮样式

<Button type="primary">Primary</Button>
<Button type="secondary">Secondary</Button>
<Button type="ghost">Ghost</Button>

```js copy
import Button from "mmixel-ui"

export default function App() {
  return (
    <Button type="primary">Primary</Button>
    <Button type="secondary">Secondary</Button>
  )
}
```

### 禁用

使用 `disabled` 禁用按钮

<Button type="primary" disabled>Primary</Button>
<Button type="secondary" disabled>Secondary</Button>

```js copy
import Button from "mmixel-ui"

export default function App() {
  return (
    <Button type="primary" disabled>Primary</Button>
    <Button type="secondary" disabled>Secondary</Button>
  )
}
```

### 加载

使用 `loaidng` 让按钮处于加载状态

<Button size="ty" loading>跑跑跑</Button>
<Button size="sm" loading>跑跑跑</Button>
<Button loading>跑跑跑</Button>
<Button type="primary" loading>跑跑跑</Button>

```js copy
import Button from "mmixel-ui"

export default function App() {
  return (
    <Button size="ty" loading>跑跑跑</Button>
    <Button size="sm" loading>跑跑跑</Button>
    <Button loading>跑跑跑</Button>
    <Button type="primary" loading>跑跑跑</Button>
  )
}
```

### Props

| Prop     | Type                                               | Default     | Desc         |
| :------- | :------------------------------------------------- | :---------- | :----------- |
| size     | `"ty"\| "sm"   \| "md"\| "lg"\| "xl"`              | `"md"`      | 按钮的大小   |
| rounded  | `boolean`                                          | `false`     | 按钮是否圆角 |
| type     | `"default" \| "primary" \| "secondary" \| "ghost"` | `"default"` | 按钮的样式   |
| disabled | `boolean`                                          | `false`     | 按钮禁用状态 |
| loading  | `boolean`                                          | `false`     | 按钮加载状态 |
| icon     | `ReactNode`                                        | `undefined` | 按钮图标     |
